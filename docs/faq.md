# 常见问题

::: details 安装后没有效果？
请确认：
1. LSPosed 中模块已启用且作用域正确勾选
2. 已重启作用域内的应用
3. HyperCeiler 中破解焦点通知白名单已开启
4. 系统版本为HyperOS3，且LSPosed API为101
5. 点击软件主页面测试按钮，如果无通知代表LSPosed模块未生效，如果是普通安卓样式通知代表破解焦点白名单未生效
:::

::: details 通知没有显示为超级岛样式？
- 确保应用发出的是**标准安卓通知**，自定义通知样式不受支持
- 检查该应用在手机自身通知设置中是否开启焦点通知权限
- 确认 HyperCeiler 破解焦点通知已正确设置
:::

::: details 通知样式改变，如何恢复？
- 在软件设置内关闭焦点通知
:::

::: details 超级岛光效不起效果？
1. 检查系统是否为安卓 16 的澎湃 OS3
2. 检查是否使用了第三方的系统界面组件
3. 第三方系统可能不受支持
:::

::: details 修改配置后不生效？
- 更新应用后需要重启作用域
- 尝试重启系统界面
:::

::: details 同时出现悬浮通知和焦点通知怎么办？
- 请尝试到系统自身通知设置中，关闭悬浮通知
:::

::: danger 出现BUG？
请上传LSPosed日志，并附带复现步骤或者完整视频，虽然大概率不会修 ):
:::
